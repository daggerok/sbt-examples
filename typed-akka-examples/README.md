# typed akka

with __java__ use:

```java
//...
import akka.actor.typed.*;
import akka.actor.typed.javadsl.*;
//...
```

with __scala__ use:

```scala
//...
import akka.actor.typed._
import akka.actor.typed.scaladsl._
//...
```

```shell script
./sbtw "project javaHello" clean run
./sbtw "project scalaHello" clean run
```
